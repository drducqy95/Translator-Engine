# 第915章 诺文的新想法

最终。
伪装身份，前往深邃地域展开秘密调查的团队人数达到了近百人的庞大规模。
雨果·西维负责了扮演奴隶商人的任务，他将是这支队伍明面上“管事”的。
培特、盖尔还有多兰他们装作雨果的手下。
布兰登、兰斯洛特、山姆这种一看就很能打的家伙则充当了商队护卫的职责。
他们是真的护卫。
不用演。
毕竟，在深邃地域那种危险的地方，很多时候，武力比道理见效更快。
雨果财大气粗地直接临时租用了金潮破风号的一次专线运输。
乘坐飞艇赶赴石堡的路上，众人逐渐聊到魔物赤潮后雷亚克王国发生的变化。
“你们知道吧，王都那边出了件大事。”
“一群疯狂的邪教徒居然敢在庇护平原举行献祭召唤仪式！”
“用来献祭的祭品来源居然是辉光神教近几年拘留囚禁的犯人！”
“啧啧啧，不仅哈里斯·尤金是个背弃了辉光信仰的渣滓，辉光神教内部肯定有很多人也出了问题，他们只是还没被发现而已！”
有人提起话题。
立马有人跟着议论起来。
“我听说，那场仪式爆发的时候，王都的守备军和狮鹫骑士团忽然得到了强化，这才让他们得以快速破坏仪式？”
“笨！不是忽然得到强化，那绝对是王室的底牌！”
有人说得信誓旦旦，语气无比笃定。
“王都斯塔罗什繁荣千年屹立不倒，一定有什么压箱底的手段，不管遇到什么危险，王室肯定都能动用底牌来解决！”
“不要小看王室，他们的底蕴可深厚着呢！”
“别忘了，先前鎏金俱乐部暴露真相的时候，国王陛下可是毫不犹豫地拿出堆成山的金子，用来置换原本属于卢恩·蒙格的资产，填平鎏金俱乐部给人们造成的损失。”
“没有足够的积累，拿得出那么多黄金吗？”
这个说法很快得到了大多数信徒的认同。
有关那场在庇护平原出现的献祭召唤仪式，虽然被及时阻止，没有让邪教徒召唤出真正危险的超阶魔物，但这无疑又是一项严重打击辉光神教声望的丑闻。
哈里斯·尤金的身败名裂，让原本曾虔诚信仰的辉光之神的普通人开始对教会产生不信任的情绪。
鬼知道那些在公开场合一副威严、神圣样子的辉光教士，他们在暗地里会干出多么龌龊的勾当！
怀疑一旦产生。
再想弥补可就难了。
尤金大主教的丑闻带来了一场席卷雷亚克王国的巨变。
肯瑞丁国王先是以极其强硬的态度，宣布接管了辉光教会的圣教军指挥权。
未来的圣教军依然由辉光神教进行培养、训练，但指挥大权将由王室负责掌管。
紧接着，与哈里斯·尤金有关的一系列辉光教士，特别是那些由他提名晋升的，统统停职，无一例外。
这些人需要等待接受由王室组织的审查，再决定他们是否能够回归原本的职责和工作。
特别是尤金大主教的长子，雷杨克·尤金。
刚赴任格林港地区主教半年多的时间，噩耗传来，他被直接免职，而且传讯要求他尽快前往斯塔罗什接受调查。      次日。
雷杨克·尤金失踪。
这一副心里有鬼、畏罪潜逃的模样，变成了辉光教会近期爆出的又一桩丑闻。
债多了不愁，虱子多了不痒。
大概是近段时间辉光教会里闹出的乱子实在太多，雷杨克·尤金的失踪甚至没能在王国里掀起多少波澜，只是有关他的通缉令被发往了大陆各地便算是告一段落。
至于那场邪教徒组织的献祭召唤仪式究竟是如何被破坏的。
知晓真相的存在少之又少。
因为有关弗拉丁王子在大法师的法师塔地下室里使用那枚“核心”的消息被进行了严格的保密封锁。
诺文对此也有着不小的兴趣和好奇。
莫克瑞德王室手里也有一份造物圣台的核心组件，这没什么好意外的。
无上真理教会的那些邪教徒都快跟批发一样的使用那些核心组件了！
同样是从旧日时代传承到如今的悠久存在，要说莫克瑞德王室手里没点类似的东西，诺文第一个不信！
令诺文感到有趣的是：
在此之前，诺文一直以为，与造物圣台有关的各类“核心”可能是因为造物圣台产生的故障而不慎流落进凡世维度，然后再被凡人发现的。
但在对这个世界有了更加深入的了解后。
诺文发现，情况没有这么简单。
“核心”组件种类繁多。
有些被信仰邪神的家伙们找到，并以相当邪门的方式使用着它们。
比如曾经差点彻底污染了瑞斯蒂的幻象核心，比如石堡造物祭坛中用于搭建那个让“使者”对矮人们完成洗脑控制的塑形核心、惩戒核心，还有那个在格林港发现的定位核心。
但有些核心组件则不然。
比如，【往日荣光】屏障这道超阶神术奇迹，其实是借助了【屏障核心】的力量才得以完成，而这枚核心被巨龙们奉为圣物。
还有属于莫克瑞德王室的那枚【征召核心】。
他们明显清楚地知道该如何使用它！
尤其是当弗拉丁王子使用征召核心时，关注着情况的诺文同样注意到了那一连串神秘出现的信息。
这就让诺文忍不住产生了一个新的、大胆的猜测：
“或许……”
“有些诞生自造物圣台的核心，它们本来就是要交给凡人使用的？”
“那位造物主宰，祂曾亲自准备了一些超越凡人力量极限的东西，然后作为底牌交给了凡人？”
“……可是祂为什么要这么做啊？”
诺文琢磨了半天，还是没什么头绪。
这个全新的猜测符合他在凡世维度中观察到的现象。
但却很难找到一个合适的理由，来解释造物主宰的意图！
祂明明已经强大到足以创造掌管超凡力量的神明，亲自塑造了一个有神明与神术存在的世界。
结果祂又给予了凡人一些几乎能让他们比肩神明的“底牌”。
“祂到底是想让凡人信仰坚定，虔诚地向神明祈祷？还是想让他们不要依赖神明，学会自食其力？”
“这不是互相矛盾嘛？”
